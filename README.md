# Task 2 - Customer Segmentation

## Objective
Group retail store customers using K-Means Clustering 
based on Annual Income and Spending Score.

## Dataset
Mall Customer Segmentation (Kaggle)

## Model
- Algorithm: K-Means Clustering
- Optimal K: 5 clusters
- Method: Elbow Method

## Libraries Used
- Python, Pandas, NumPy
- Scikit-learn, Matplotlib# PRODIGY_ML_02
